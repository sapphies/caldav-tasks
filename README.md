# caldav app test
a work in progress caldav compatible task management client for desktop

> [!WARNING]  
> the app is currently in alpha so you might encounter bugs here and there. if you do, [file a bug in the "issues" page](https://github.com/sapphies/caldav-tasks/issues) and let us know
